# My Next Blog

## 🖥️ 프로젝트 소개

Next.js 기반 개인 블로그입니다. 프로젝트 진행과정이 기록되어있습니다.

## 📅 개발기간

25.06.

## 👥 개발 인원

개인 프로젝트

## ⚙️ 주요 기능

- mdx 파싱을 이용한 블로그 게시물
- 다크모드 & 시스템모드 지원
- 각 카테고리 & 태그 별 게시물 분류
- 스크롤에 반응하는 ToC, 스크롤 프로그레스 바
- Giscus를 이용한 댓글 기능
- 반응형 UI를 통한 모바일 뷰포트 지원
- OpenGraph를 이용한 메신저 공유 지원
- SSG를 이용한 SEO향상으로 검색엔진 노출도 상승
- 게시물 페이지네이션
